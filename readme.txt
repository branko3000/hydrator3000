Aufgaben:
Zu jeder Aufgabe gehört eine Skriptdatei. Diese findest du im Ordner /js.
Jede Skriptdatei enthält eine Beispiellösung für die vorhergegangene Aufgabe.
1. Schreibe eine Funktion, die den Inhalt des Fasses um eine gegebene Menge erhöht.
2. Schreibe eine Funktion, die die Anzeige aktualisiert.
   https://developer.mozilla.org/de/docs/Web/API/Document/getElementById
3. Schreibe eine Funktion, die den Wert der Fass Variable speichert.
   https://developer.mozilla.org/de/docs/Web/API/Window/localStorage
4. Schreibe eine Funktion, die einen Etikettstring basierend auf dem heutigen Datum erzeugt und ihn in einer Variable abspeichert.
   https://developer.mozilla.org/de/docs/Web/JavaScript/Reference/Global_Objects/Date
5. Ändere die Speicherfunktion so, dass sie das Fass mit dem Etikett abspeichert
6. Überprüfe, ob es bereits ein Fass unter dem aktuellen Etikett gibt, erzeuge ein neues wenn nicht, ändere den Wert der Fassvariable wenn doch.
   https://developer.mozilla.org/de/docs/Web/JavaScript/Reference/Statements/if...else

Herausforderungen:
- Wenn du bereits weißt, wie man Grafiken mit einem Computer erstellt oder bearbeitet, zeichne deine eigenen Grafiken für Gläser und Flasche.
- Wenn du das noch nicht weißt, lerne mit GIMP oder Inkscape umzugehen, beide Programme sind kostenlos und Open Source.
  https://www.gimp.org/
  https://inkscape.org/de/
- Füge mehr Gefäße hinzu.
- Lerne ein wenig CSS und gestalte das User Interface etwas schöner.
  https://developer.mozilla.org/en-US/docs/Learn/CSS/First_steps
- Schreibe ein wenig Code und Füge eine Schaltfläche hinzu, mit der man die getrunkene Menge frei eingeben kann.
  https://developer.mozilla.org/de/docs/Web/API/Window/prompt
- Schreibe noch ein wenig mehr Code und bastele eine Statistik Funktion, mit der man auf die vergangene Woche zurückblicken kann.
- Benutze Surge um deine Applikation im Internet bereit zu stellen und sie dann zum Beispiel zum Homescreen deines Handys hinzuzufügen (oder verwende einen anderen statischen Hosting Service)
  http://surge.sh/
